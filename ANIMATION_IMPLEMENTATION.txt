🚀 NEXT STEPS TO FULLY ACTIVATE

Step 1: Add State Management to App.tsx
────────────────────────────────────────

  Add these states:
    const [activeChipAnimation, setActiveChipAnimation] = useState(null);
    const [activeActionBox, setActiveActionBox] = useState(null);
    const [playerLastActions, setPlayerLastActions] = useState({});


Step 2: Trigger on Player Actions
──────────────────────────────────

  In your handleAction function:

    const handlePlayerAction = (seat, action, amount) => {
      // Show action box
      setActiveActionBox({
        action: action,
        amount: amount,
        playerName: players[seat].name,
        seat: seat
      });

      // Animate chips if betting
      if (amount > 0) {
        setActiveChipAnimation({
          amount: amount,
          fromSeat: seat,
          isAllIn: action === 'allin'
        });
      }

      // Update player state for glow
      setPlayerLastActions({
        ...playerLastActions,
        [seat]: action
      });
    };


Step 3: Render in App.tsx
─────────────────────────

  Add to your render:

    import ChipAnimation from './components/ChipAnimation';
    import ActionBox from './components/ActionBox';

    {/* Chip animation overlay */}
    {activeChipAnimation && (
      <ChipAnimation
        {...activeChipAnimation}
        onComplete={() => setActiveChipAnimation(null)}
      />
    )}

    {/* Action box */}
    {activeActionBox && (
      <ActionBox
        {...activeActionBox}
        onComplete={() => setActiveActionBox(null)}
      />
    )}


Step 4: Pass Player States to Table
────────────────────────────────────

  Update Table component to receive lastAction:

    <Table
      players={players.map(p => ({
        ...p,
        lastAction: playerLastActions[p.seat]
      }))}
      // ... other props
    />
