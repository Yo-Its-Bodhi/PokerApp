## 🔧 Customization

### Change Animation Speed
Edit `web/src/index.css`:
```css
.card-flipper {
  transition: transform 0.6s;  /* ← Adjust flip speed */
}

@keyframes dealCard {
  /* Change 0.5s duration */
}
```

### Change Card Sizes
Edit Card component props:
```tsx
size="small"   // 14×20px (hole cards)
size="medium"  // 20×28px (community)
size="large"   // 24×32px (special)
```

### Add Sound Effects
Add to Card component:
```tsx
const dealSound = new Audio('/sounds/card-deal.mp3');
useEffect(() => {
  dealSound.play();
}, []);
```
