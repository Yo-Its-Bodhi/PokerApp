   Added State:
   const [revealedCards, setRevealedCards] = useState<number>(0);
   
   Purpose:
   Tracks how many community cards should be revealed (0-5)
   
   
   Updated startDemoGame Callback:
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   
   Before:
   setCommunityCards(gameState.communityCards);
   
   After:
   setCommunityCards(prevCards => {
     if (newCommunityCards.length > prevCards.length) {
       // New cards dealt - start face-down
       setRevealedCards(0);
       // Then reveal after 100ms
       setTimeout(() => setRevealedCards(newCommunityCards.length), 100);
     } else if (newCommunityCards.length === 0) {
       // New hand - reset
       setRevealedCards(0);
     }
     return newCommunityCards;
   });
   
   Flow:
   1. Community cards increase (e.g., 0 → 3 for flop)
   2. Set revealedCards to 0 (all face-down)
   3. Update communityCards state
   4. Wait 100ms
   5. Set revealedCards to 3 (trigger flip animations)
   
   
   Pass to Table:
   <Table revealedCards={revealedCards} ... />

   After:

   <Card
     suit={cardData.suit}
     rank={cardData.rank}
     color={cardData.color}
     size="medium"
     animationDelay={i * 0.15}
     faceDown={!isRevealed}        // ← Controls if back is shown
     showFlipAnimation={isRevealed}  // ← Triggers flip animation

   Logic:
   - Card index < revealedCards → isRevealed = true
   - isRevealed = true → faceDown = false, showFlipAnimation = true
   - isRevealed = false → faceDown = true, showFlipAnimation = false
   
   Example (Flop with 3 cards, revealedCards = 2):
   • Card 0: i=0 < 2 → isRevealed=true → Flips and shows face
   • Card 1: i=1 < 2 → isRevealed=true → Flips and shows face
   • Card 2: i=2 < 2 → isRevealed=false → Stays face-down
