⚙️ IMAGE SPECIFICATIONS


Perfect Image Settings:
  ✓ Format:      PNG with alpha channel (transparency)
  ✓ Size:        512×512px minimum (1024×1024 recommended)
  ✓ Background:  Transparent (no white/gray)
  ✓ Colors:      Blue tones (cyan, navy, royal blue)
  ✓ Style:       Geometric, minimalist design
  ✓ Contrast:    High contrast for visibility

Bad Image Settings:
  ✗ Format:      JPG (no transparency)
  ✗ Size:        Too small (<256px, will look pixelated)
  ✗ Background:  White or colored (won't blend)
  ✗ Colors:      Red/green (doesn't match theme)
  ✗ Style:       Too complex/detailed at small size


Remove Background:
  → If your image has a background, visit:
  → https://www.remove.bg/
  → Upload image → Download PNG with transparency

🔧 TROUBLESHOOTING


Image Not Showing?

  Problem:  Card shows geometric pattern instead of icon
  
  Check #1: Filename must be exactly "card-back-icon.png"
            └─ No spaces, capital letters, or extra words
  
  Check #2: File must be in correct folder
            └─ C:\Users\dj_ba\Desktop\Poker\web\public\
  
  Check #3: Clear browser cache
            └─ Press Ctrl + Shift + Delete
            └─ Or hard refresh: Ctrl + F5
  
  Check #4: Verify file path in browser
            └─ Open: http://localhost:5177/card-back-icon.png
            └─ Should show your image


Image Looks Blurry?

  Solution: Use higher resolution image
            └─ Current: 256×256 → Change to: 1024×1024
            └─ Re-export from design software at higher DPI


Wrong Colors?

  Note: Card back has blue gradient overlay
        └─ Your icon blends with this gradient
        └─ Use white/light icon for best visibility
        └─ Or edit Card.tsx to change gradient colors


Animation Not Smooth?

  Check: Browser performance
         └─ Close other tabs
         └─ Update graphics drivers
         └─ Enable hardware acceleration in browser
