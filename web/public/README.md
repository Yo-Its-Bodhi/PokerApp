# Card Back Image

## How to Add Your Custom Card Back

1. **Save your blue icon image as**: `card-back-icon.png`
2. **Place it in this folder** (web/public/)
3. **Refresh the browser** to see it on all card backs

## Image Requirements

- **Format**: PNG with transparent background
- **Size**: 512x512px or larger (recommended)
- **Style**: Blue geometric/icon design
- **Background**: Transparent (alpha channel)
