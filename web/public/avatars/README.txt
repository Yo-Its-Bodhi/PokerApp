AVATAR IMAGES FOLDER
====================

To add the neon heart avatar:
1. Save the neon heart with lightning bolt image as: neon-heart.png
2. Make sure the image has a transparent background (no white circle)
3. Recommended size: 256x256 or 512x512 pixels

The system will automatically use this image when users select the first avatar in the "special" category.

Format: PNG with transparency
Location: /web/public/avatars/neon-heart.png
